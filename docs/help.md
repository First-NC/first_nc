# First NC 帮助文档

## 快速开始

1. 启动应用。
2. 点击“打开 NC”选择文件。
3. 建议优先使用 `demo_nc/` 下的示例文件验证功能。

## 文件与编辑

- 支持打开：`.nc`、`.anc`
- 支持搜索、排序和最近文件
- 支持保存与另存为

## 3D 视图交互

- 左键拖动：平移
- 右键拖动：旋转
- 滚轮：缩放
- `F`：回中心并完整适配
- `G`：显示或隐藏网格

## 常见问题

### 双击文件无法打开

- 确认系统已将 `.nc/.anc` 默认打开方式设置为 First NC
- 确认当前运行的是桌面应用而不是浏览器开发页
