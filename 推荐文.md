## 激光双面模式：传统工具做不到的事

这是 First NC 最大的差异化能力。

激光雕刻机做双面加工，程序里用两套坐标：正面 `XYZ`，反面 `UVW`。传统模拟器要么不解析 UVW，要么当成错误指令跳过。

First NC 的处理方式不一样：**文件拖进来，自动检测有没有 UVW 指令。有，自动进入激光模式，正反面同时显示。**

![激光模式：正面 XYZ 刀路与反面 UVW 路径（紫色）渲染在同一 3D 场景中](https://cdn.jsdelivr.net/gh/First-NC/first_nc@main/screenshots/%E6%BF%80%E5%85%89.png)

- 不需要手动切换模式，拖文件即识别
- 反面路径用紫色独立着色，图例标注「UVW 反面路径」
- 正反面完全联动：编辑器改代码、拖进度条、3D 拾取都双向同步
- 之前对不准废掉的板材，现在看一眼就知道

## 像看视频一样校验刀路

NC 程序上百行甚至上千行，靠肉眼逐行检查不现实。

First NC 的播放控制跟视频播放器一样直观：

- **播放 / 暂停** — 从头跑到尾，看刀路整体走向
- **单步前进 / 后退** — 一行一行过，拐角、下刀点重点看
- **三档速度** — 低速细看拐角，中速通览，快速扫一遍确认无异常
- **进度条拖拽** — 直接跳到想检查的位置
- **跟随刀位** — 3D 视角自动追着刀头走，不会跟丢

编辑器和 3D 视图是双向联动的：在 3D 视图里点一个路径点，编辑器自动定位到对应代码行。在编辑器里改代码，3D 图跟着刷新。

## 跨平台 + 6MB：不用再伺候虚拟机了

传统 NC 模拟器几乎都是 Windows 专属。Mac 用户想用？装 Parallels（花钱、占空间）。Linux 工控机？很少有选择。

First NC 三个平台都有原生安装包，不用编译、不用 Wine、不用虚拟机。每出一个新版本，三个平台同步发布。

| 系统 | 安装包大小 |
|---|---|
| Windows 10+ | 6.8 MB |
| macOS（Intel / Apple Silicon） | 9.2 MB |
| Linux（Ubuntu / Debian）24+ | 8.8 MB |

打开就能用，深色主题默认就好看，支持多文件同时编辑与查看：

切换到沉浸式全屏 3D 模式，刀路清晰锐利，色彩分层一目了然：
