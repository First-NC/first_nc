# 写了几百行 G 代码，你敢直接上机吗？

一块多层实木板，正反面各跑一套 NC 程序。正面 XYZ 刻完了，翻面 UVW 走反面。文件一换，坐标系一乱——整块板材报废。

这不是段子。做数控的都懂这种痛：**程序没报错，刀路有问题，但你不敢赌。**

以前怎么办？打印代码，荧光笔逐行比对。或者装个仿真软件，Windows 独占，Mac 和 Linux 用户先装虚拟机。再或者，硬着头皮直接上机，开空跑盯着刀尖看。

**First NC 让这件事变得简单了。** 免费、开源、跨平台，装上就能用。 **NC 代码不再是天书——它看得见、走得了、能验证。**

## 🚀 6MB 的安装包，三平台原生支持

你没看错。不是 600MB，是 **6MB**。

| 平台 | 安装包大小 |
|---|---|
| 🪟 Windows | **6.8 MB**（EXE / MSI） |
| 🍎 macOS | **9.2 MB**（DMG） |
| 🐧 Linux | **8.8 MB**（DEB） |

对比一下：Electron 构建的同类型工具安装包通常在 150~300MB，First NC 用的 **Tauri 架构**，安装包小了一个数量级，启动速度也甩开几条街。

而且不是「先有 Windows 版再移植」——从一开始就是三平台齐头并进，一套代码编译 Win / Linux / Mac 三个原生安装包，CI 自动构建。

从 v1.0.0 到 v1.2.5 迭代了 **30 个版本**，85 个 TypeScript 模块、692 KB 前端源码，半年持续打磨，不是随便攒出来的玩具。

**还有这些实用能力，值得一提：**

- 📂 原生 `.nc` / `.anc` 文件，自动识别 UTF-8 / GBK 编码
- 🌓 四种主题：浅色 / 深色 / 藏蓝 / 跟随系统
- 🌐 中英文一键切换，默认中文
- ✏️ Monaco 编辑器（VS Code 同款），代码折叠、语法高亮、搜索替换全有
- 🔄 最近 10 个文件自动恢复
- 📦 应用内增量更新
