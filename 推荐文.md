## ⌨️ 快捷键一览：熟手可以完全不用鼠标

First NC 提供了丰富的快捷键，操机师傅上手极快：

| 快捷键 | 功能 |
|---|---|
| `左键拖拽` | 旋转 3D 视角 |
| `中键拖拽` | 平移 3D 视图 |
| `滚轮` | 缩放 3D 视图 |
| `F` | 回中心适配（按一下视角复位） |
| `G` | 切换背景网格显示 |

按 `G` 键可以快速开关网格，关闭后视图更干净，便于截图和观察细节。按 `F` 键一键回中心，视角自动复位：

![快捷键演示：G 键切换网格、F 键回中心复位视角](https://cdn.jsdelivr.net/gh/First-NC/first_nc@main/screenshots/gif_grid_focus.gif)

## 🚀 6MB 的安装包，三平台原生支持

你没看错。不是 600MB，是 **6MB**。

| 平台 | 安装包大小 |
|---|---|
| 🪟 Windows | **6.8 MB**（EXE / MSI） |
| 🍎 macOS | **9.2 MB**（DMG） |
| 🐧 Linux | **8.8 MB**（DEB） |

对比一下：Electron 套壳的软件动辄两三百 MB，First NC 用的 **Tauri 架构**，安装包小了一个数量级，启动速度也甩开几条街。

而且不是「先有 Windows 版再移植」——从一开始就是三平台齐头并进，一套代码编译 Win / Linux / Mac 三个原生安装包。

从 v1.0.0 到 v1.2.5 迭代了 **30 个版本**，85 个 TypeScript 模块、692 KB 前端源码，半年持续打磨，不是随便攒出来的玩具。

**还有这些实用能力，值得一提：**

- 📂 原生 `.nc` / `.anc` 文件，自动识别 UTF-8 / GBK 编码
- 🌓 四种主题：浅色 / 深色 / 藏蓝 / 跟随系统
- 🌐 中英文一键切换，默认中文
- ✏️ Monaco 编辑器（VS Code 同款），代码折叠、语法高亮、搜索替换全有
- 🔄 最近 10 个文件自动恢复
- 📦 应用内增量更新
