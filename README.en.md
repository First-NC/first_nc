# First NC

[中文 README](./README.md)

First NC is a cross-platform desktop application for NC viewing, editing, and 3D simulation.

## Quick Start

- Try the sample files in `demo_nc/` first.
- Open `.nc/.anc` files from `demo_nc` to validate linkage, simulation, grid, and picking.

## Featured Capability

### Laser Integrated Machine Mode

Laser Integrated Machine mode is the most distinctive capability in First NC. It is designed for dual-side machining programs and synchronized verification.

- Front-side coordinate domain: `XYZ`
- Back-side coordinate domain: `UVW`
- Dedicated UVW path color in the 3D legend
- Full linkage with editor, progress, and 3D picking
- User mode preference persistence across restarts

## Core Capabilities

- Open `.nc/.anc` files
- Linked file list, code editor, and 3D viewport
- Play, pause, step, speed, and progress dragging
- View presets, pan, rotate, zoom, and refocus (`F`)
- Grid toggle (`G`)
- Runtime i18n
- Multiple themes
- Recent file restore
